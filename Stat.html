Tests Statistiques

Test statistique

En statistiques, un test d'hypothèse est une démarche consistant à rejeter ou à ne pas rejeter (rarement accepter) une hypothèse statistique, appelée hypothèse nulle, en fonction d'un jeu de données (échantillon). Il s’agit de statistique inférentielle : à partir de calculs réalisés sur des données observées, nous émettons des conclusions sur la population, en leur rattachant des risques de se tromper.

Plus d'informations

Dans le cadre de ce projet, nous avons réalisés de deux types de sorties différents.

  1. Dans le premier fichier, l'univers est une matrice carrée de taille 30. La probabilité des cellules contenu dans l'univers était de 10%. La probabilité d'énergie au départ de 100%. La probabilité était de 0. Les données utilisées sont celles de la 150eme générations. L'univers ne se renouvelle pas en énergie
  2. Dans le deuxième fichier, l'univers est aussi une matrice carrée de taille 30. La probabilité des cellules contenu dans l'univers était de 10%. La probabilité d'énergie au départ de 1%. La probabilité des virus était de 0. Les données utilisées sont celles de la 150eme générations. L'univers se renouvelle en énergie

Le clustering est une méthode très intéressante pour étudier la structure et le groupement de nos données. Il est généralement basé sur la similarité ou la distance entre les objets.

Dans notre cas, nous avons utilisés Le partitionnement en k-moyennes qui est une méthode de partitionnement de données et un problème d'optimisation combinatoire. Ici le nombre de k a été fixé a 4. Dans cette étude, la probabilité de déplacement de plusieurs centaines de cellules vont être mesurée.

Cluster sur le fichier 1


Nos données possèdent une structure hiérarchique c’est-à-dire que nos groupes possèdent des sous-groupes mutuelles. Ceci va être visualisé dans une structure appelé dendrogramme qui présente un chemin intuitif pour visualiser la structure hiérarchique.

ClusPlot sur le fichier 1


Cette figure montre la classification de ces 4 clusters, les cellules sont regroupés en 4 clusters en fonction de leurs probabilité de déplacement. Ces données ont été obtenir à partir du package mclust.
